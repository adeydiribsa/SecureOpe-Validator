# InfraGuard

Lightweight Continuous Security Validation Platform for Infrastructure & Server Security Monitoring

Automates repetitive security checks including:
- Open port exposure validation
- CIS-aligned configuration checks
- Server misconfiguration detection
- Security control visibility
- Automated PDF reporting

Most organizations still rely on periodic manual security reviews.

This project was created to help shift operational security from fragmented manual checks toward continuous and repeatable security validation.
